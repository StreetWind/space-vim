## Description

This layer adds support for the painless text alignment.


## Install

To use this configuration layer, add it to your `~/.spacevim`.

## Dependencies
    ## Ctags
        ```C++
        Ubuntu
            sudo apt-get install ctags
        ```
    ## Cscope
        ```C++
        Ubunt
            sudo apt-get install cscope
        ```
